# Vue3后台商城管理系统

## 项目介绍

后台商城管理系统，功能还是挺多的，用于练手项目，觉得还不错的小伙伴记得点个start哦~ 

## 技术栈
+ Vue3 
+ Vite 
+ ElementPlus 
+ Echarts
+ Axios 
+ WindiCSS
